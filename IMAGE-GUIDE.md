## üöÄ Image Optimization (IMPORTANT!)

Large images slow down your site. Optimize before uploading!

### Online Tools (Easy):
1. **TinyPNG** (https://tinypng.com)
   - Drag and drop images
   - Automatically optimizes
   - Download compressed versions

2. **Squoosh** (https://squoosh.app)
   - More control over compression
   - Side-by-side comparison
   - Free, by Google

3. **ImageOptim** (Mac only, https://imageoptim.com)
   - Desktop app
   - Drag and drop
   - Automatic optimization

### Target File Sizes:
- Hero images: Under 500KB
- Gallery images: Under 200KB
- Project images: Under 300KB

## üìù Adding More Images Later

### To add a new homepage image:
1. Add image to `/images/` folder
2. Open the HTML file where you want to use it
3. Find similar `<img>` tag
4. Update the `src` attribute: `src="images/your-new-image.jpg"`

### To add a new project:
1. Add image to `/images/projects/` folder
2. Open `projects.html`
3. Copy an existing project block:
```html
<div class="project-item">
    <div class="project-image">
        <img src="images/projects/your-project.jpg" alt="Project Description">
    </div>
    <h3 class="project-title">Your Project Name</h3>
</div>
```
4. Paste it in the projects grid
5. Update the filename and title

## üí° Pro Tips

### Consistency
- Use the same photo editing style across all images
- Keep white balance consistent (warm tones, not too cool)
- Maintain similar composition styles

### Mobile Optimization
- Test how images look on mobile devices
- Avoid text in images (hard to read on small screens)
- Ensure focal points are centered

### Alt Text
The `alt` attribute in your HTML describes images for:
- Screen readers (accessibility)
- SEO (search engines)
- When images fail to load

Always make them descriptive:
- ‚ùå Bad: `alt="image1"`
- ‚úÖ Good: `alt="Bohemian living room with layered textiles and floor pillows"`

## üÜò Troubleshooting

### Image not showing?
1. Check the file path is correct
2. Ensure filename matches exactly (case-sensitive!)
3. Verify file is actually in the folder
4. Try refreshing the page (Cmd/Ctrl + Shift + R)

### Image looks blurry?
- Upload a higher resolution version
- Minimum 1920px for hero images
- Use quality setting of 80-90% when optimizing

### Image loads slowly?
- File is too large, optimize it
- Target under 500KB for hero, under 200KB for others
- Use TinyPNG or Squoosh

### Wrong aspect ratio?
- Crop image to match recommended dimensions
- Use CSS to adjust if needed (ask for help)
